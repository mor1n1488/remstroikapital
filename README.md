# Ремстройкапитал

Проект переработан из одного HTML-файла в структуру:

- `public/index.html`
- `public/styles.css`
- `public/app.js`
- `server.js`
- `data/remstroikapital.db` — создаётся автоматически при первом запуске

## Что изменено

1. CSS вынесен в отдельный файл.
2. JavaScript вынесен в отдельный файл.
3. Убран дублирующийся блок калькулятора.
4. Добавлен отсутствующий раздел `О компании`, на который ссылалось меню.
5. Локальное хранение пользователей, отзывов, заявок и цен заменено на SQLite.
6. Добавлены API-маршруты для:
   - регистрации
   - входа
   - отзывов
   - заявок
   - изменения цен
7. Корзина оставлена в `localStorage`, так как это клиентское временное состояние. При отправке корзины создаётся серверная заявка.

## Запуск

```bash
npm install
npm start
```

Открыть в браузере:

```bash
http://localhost:3000
```

## Примечание

Пароли в этой учебной версии хэшируются через SHA-256 без соли. Для реального продакшена лучше заменить на `bcrypt` и добавить полноценную авторизацию через сессии или JWT.
